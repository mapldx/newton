Get Data

Description: This endpoint retrieves data about a person.

URL: https://api-v2.example.com/api/data

Method: GET

Request

None: No request body required for this endpoint.

Response

name: The name of the person.

age: The age of the person.

city: The city where the person resides.

Response Types

200: Success. Data about the person is returned.

500: Internal Server Error. Returned if an exception occurs.

Request Examples

cURL Example


curl -X GET https://api-v2.example.com/api/data

Python Example


import requests
response = requests.get('https://api-v2.example.com/api/data')
print(response.json())

Node.js Example


const axios = require('axios');
axios.get('https://api-v2.example.com/api/data')

    .catch(error => console.error(error));

{

Create User

Description: Endpoint to create a new user.

URL: https://api-v2.example.com/api/users

Method: POST

Request

username: The username of the new user.

password: The password of the new user.

Response

message: A message indicating the result of the operation.

Response Types

201: User successfully created.

401: Unauthorized access.

Request Examples

cURL Example


curl -X POST https://api-v2.example.com/api/users -d 'username=johndoe&password=pass123'

Python Example


import requests

url = 'https://api-v2.example.com/api/users'
payload = {'username': 'johndoe', 'password': 'pass123'}
response = requests.post(url, data=payload)
print(response.json())

Node.js Example


const axios = require('axios');

axios.post('https://api-v2.example.com/api/users', {
  username: 'johndoe',
  password: 'pass123'
})
.then(response => {
  console.log(response.data);
})
.catch(error => {
  console.error(error);
});

Response Example


{
  "message": "User created"
}
